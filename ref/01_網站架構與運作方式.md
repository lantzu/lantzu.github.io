# 網站架構與運作方式 (Dawson 風格 - 莫蘭迪淺紫)

本文件詳細描述 `lantzu.github.io` 的網站架構、設計理念以及自動化工作流。

## 1. 設計理念
參考 WordPress [Dawson](https://wordpress.com/zh-tw/theme/dawson) 主題，本網站採用 **視覺優先 (Visual-First)**、**編輯感 (Editorial Minimalism)** 與 **優雅細緻** 的設計風格：
- **視覺風格**：大膽的封面圖、優雅的格狀排版、注重內容呈現。
- **配色方案 (莫蘭迪淺紫)**：
  - 背景：`#F3F0F7` (極淡的低飽和紫灰色，營造柔和感)
  - 主要文字：`#4E4B59` (深炭紫，保持高對比度但不生硬)
  - 輔助色/邊框：`#DED8E5` (淡薰衣草紫)
  - 強調色：`#A69CB7` (灰調紫羅蘭)
- **字體選用**：
  - 標題：`Cormorant Garamond` (襯線體 Serif)，展現經典編輯感與優雅。
  - 內文：`Montserrat` 或系統預設無襯線體 (Sans-serif)，確保易讀性。
- **佈局**：響應式兩欄或三欄網格 (Grid)，首頁強調 Portfolio 展示感。

## 2. 網站架構 (Sitemap)
- **首頁 (Home)**：採用 Dawson 風格的 Portfolio 網格展示文章，每篇文章以大圖搭配標題與分類。
- **文章分類頁 (Category/Archive)**：依據文章屬性 (如：旅遊、生活) 進行分組展示。
- **文章內頁 (Single Post)**：全寬封面圖、優雅的標題排版、主文內容、以及嵌入的照片。
- **關於 (About)**：極簡個人介紹。

## 3. 目錄結構
```text
lantzu.github.io/
├── index.html          # 網站首頁
├── css/                # 樣式表 (Dawson & Morandi Style)
├── js/                 # 網站邏輯 (過濾、動畫)
├── assets/             # 網站靜態資源 (Icon, Logo 等)
├── posts/              # 生成後的文章 HTML 頁面
├── photos/             # 生成後供網頁讀取的照片
└── ref/                # 原始資料夾 (工作流核心)
    ├── 00_主架構.txt    # 原始需求說明
    ├── 01_網站架構...md # 本文件
    ├── categories/     # 文章分類目錄
    │   └── [category]/
    │       ├── article_v1.txt  # 原始文字檔
    │       └── image1.jpg      # 原始照片檔
    └── global_config.json      # 網站全域設定
```

## 4. 自動化工作流 (Workflow)
為了簡化維護，我們採用「資料夾驅動」的生成方式：

### 步驟 A：撰寫內容 (User 操作)
1. 在 `ref/categories/[分類名稱]/` 下建立一個 `.txt` 檔案。
2. 在文字檔中撰寫內容，並使用特定語法標註照片位置（例如：`[[photo: image1.jpg]]`）。
3. 將對應的照片放入同一個目錄中。

### 步驟 B：生成網頁 (Gemini 操作)
1. **掃描**：讀取 `ref/` 下的所有分類與文字檔。
2. **解析**：解析 `.txt` 中的標題、日期、內容與照片標記。
3. **渲染**：根據 Dawson 風格的 HTML 模板，生成 `index.html` (更新列表) 與 `posts/[文章名稱].html`。
4. **配色注入**：自動應用莫蘭迪淺紫配色方案。

### 步驟 C：發佈 (User 操作)
- 確認生成結果無誤後，執行 `git add`, `commit` 與 `push` 到 GitHub。
